# **App Name**: GlowMatch AI

## Core Features:

- AI Face Analysis Tool: Uses AI as a tool to analyze uploaded selfies for skin tone and facial structure to determine optimal beauty treatments.
- Smart Recommendations: AI-generated personalized beauty recommendations including hairstyles, skin care, and cosmetic services based on analysis data.
- Nearby Salon Discovery: A filterable UI listing beauty salons, using device geolocation to display precise distance metrics.
- Booking Workflow: Streamlined booking interface allowing users to select services, dates, and times for salon visits.
- Glassmorphic Salon Feed: High-end visual layout utilizing glassmorphism effects for salon cards and list elements for a modern feel.

## Style Guidelines:

- Primary color: Vibrant Orchid (#bd1adb). This hue reflects a bold digital aesthetic while staying within the pink-purple beauty spectrum.
- Background color: Mist Lavender (#faf7fb). A heavily desaturated, very light version of the primary to maintain the light mode theme with a soft touch.
- Accent color: Electric Violet (#6033cc). An analogous hue used for interactive states and focus elements to provide depth against the primary.
- Font pairing: 'Poppins' for clean, fashionable headlines and 'Inter' for objective, highly-readable body text.
- Responsive layout featuring modern card-based components with glassmorphism—frosted glass overlays, subtle shadows, and thin white borders.
- Fluid, soft-ease transitions for gradient backgrounds and page navigation to mirror a luxury beauty experience.
- Feather-light stroke icons with purple-to-pink gradients for feature sets and service categories.
